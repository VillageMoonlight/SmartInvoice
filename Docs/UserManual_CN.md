
# 用户手册 (User Manual)

## 1. 系统入门

### 1.1 首次登录
- 系统第一次运行无需账号。
- 首位注册的用户将**自动获得管理员权限**。
- 后续注册的用户需联系首位管理员在“账号管理”中手动点击“激活”方可登录。

### 1.2 配置 AI 引擎
- 登录后，点击左侧导航栏的 **[模型参数]**。
- **默认模式**: 使用内置的 Gemini 3.0 Flash（需在部署时配置环境变量）。
- **自定义模式**: 您可以切换到“硅基流动”或其他 OpenAI 兼容接口。输入 API Key 和模型 ID（如 `zai-org/GLM-4.6V`）后点击保存。

## 2. 发票采集

1. 点击左侧 **[智能采集]**。
2. 点击中间的上传区域，或直接拖拽文件（支持多选）。
3. 系统将展示处理进度：
   - **读取中**: 正在将文件转化为 AI 可理解的数据。
   - **识别中**: AI 正在提取金额、税率、供应商等信息。
   - **成功/失败**: 完成后会自动跳转至台账页。

## 3. 台账管理

### 3.1 查看与搜索
- 在 **[发票台账]** 页面，您可以点击发票行左侧的箭头查看完整明细（包括购方税号、备注等）。
- 使用顶部的搜索框可根据发票号码、供应商或项目名称进行实时过滤。

### 3.2 附件预览
- 点击操作列的 **蓝色图标**。
- **图片**: 在弹出层直接查看。
- **PDF**: 将在浏览器新窗口打开，您可以直接调用浏览器的打印功能。

### 3.3 数据导出
- 点击右上角的 **[导出全字段台账]**，系统将下载一份包含了 20 多个财务字段的 CSV 文件，可直接用 Excel 打开进行后续入账操作。

## 4. 常见问题排查

- **识别提示 400 错误**: 通常是由于图片过大或 API 地址配置错误，请检查“模型参数”设置。
- **发票号码识别不全**: 确保发票右上角没有被遮挡，建议使用 300dpi 以上的扫描件。
- **无法预览 PDF**: 请检查浏览器是否拦截了弹出窗口，点击“允许本站弹出窗口”即可。
